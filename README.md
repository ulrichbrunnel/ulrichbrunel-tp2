## decouvert cuisine japonaise
### Description
La cuisine japonaise est réputée pour sa variété, sa fraîcheur et son équilibre nutritionnel. Une alimentation saine repose sur une harmonie entre les différents nutriments essentiels dont le corps a besoin pour fonctionner de manière optimale. Ces nutriments se répartissent en macronutriments — glucides, protéines et lipides — et en micronutriments — vitamines et minéraux — présents dans les ingrédients traditionnels japonais comme le riz, le poisson, les légumes, le soja et les algues.
