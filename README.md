Site Mono Page – Cuisine Japonaise
Description

Ce projet est un site web mono page présentant la cuisine japonaise.
Il inclut les sections suivantes :

Accueil : introduction à la cuisine japonaise avec un texte descriptif et un bouton d’inscription.

Inscription : formulaire de contact avec différents types de champs.

Tableau : présentation des plats japonais et de leurs informations nutritionnelles.

Articles : articles détaillés sur les plats japonais avec images et mise en page en colonnes.

Pied de page : informations de contact, liens importants et réseaux sociaux.

Le site utilise HTML5, CSS3, et Flexbox pour la mise en page.


Technologies utilisées

HTML5

CSS3

Flexbox
